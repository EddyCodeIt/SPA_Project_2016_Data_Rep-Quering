I built this website to share my experience in developing on Python Microframework Flask.

The goal of developing this application was to learn what Flask is good at, why people using it. To learn about Flask toolkit and what official documentation recommends to blend it with. Find out how flexible Flask is in managing inconsistent, repeatable code and content.To decide if I would use it for a larger scale projects.


What is Flask?

From my experience of using Flask, I understood that it is a backbone of the web application that provide means for
user - server communication, i.e., handling requests from users and sending back responses, passing, modifying or deleting data & information, as well as, laying out routes on your website, and many more...
