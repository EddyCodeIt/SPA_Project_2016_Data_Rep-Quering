I built this website to share my experience in developing on Python Microframework Flask.

The goal of developing this application is to learn what utilities Flask can provided to enhance smooth programming flow, how convinient to use it and what Flask is good at, and why people using it.


What is Flask?

From my experience of using Flask, I understood that it is a backbone of the web application that provide means for
user - server communication, i.e., handling requests from users and sending back responses, passing data & informatione, as well as, laying out routes on your website, and many more...

What Flask offers and what I used:

  • Official: Jinja2 Templates Flask: Jinja2 Setup
  • Routing of a website

  • URL building methods

  • HTTP Methods

  • SQLAlchemy DB toolkit
  • Flask Login

    • From outside Flask's scope I used:

  • WTForms
  • Bootstrap Library
